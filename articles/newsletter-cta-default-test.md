# Newsletter CTA with Default Message

This article demonstrates using the newsletter CTA with the default title and description.

## How It Works

When you use just `#newsletter-cta` without parameters, the CTA will display with these default values:

- **Title**: "Stay Updated with Fresh Insights"
- **Description**: "Get the latest tech news, architecture patterns, and software craftsmanship tips delivered to your inbox. Join hundreds of developers who stay ahead of the curve."

## Flexibility

If you want custom messaging, you can also use the parametrized format:

```
#newsletter-cta('Your Custom Title', 'Your custom description text here')
```

This is useful for articles where you want a more specific call-to-action that matches the article's topic or tone.
