Remix 3 beta preview is here. Try it out now: npx remix@next new my-remix-app

This is still pre-release software, so we do not recommend it for production yet. But it is ready for experiments, demos, prototypes, and feedback from curious builders.

This beta preview is a real step in that direction: one remix umbrella for routes, request handlers, middleware, sessions, auth, forms, uploads, assets, data, UI, tests, and more.

The goal is simple: install Remix and start building, without spending the first day stitching together a dozen unrelated dependencies.
