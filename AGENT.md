## Testing

- Write tests for new features
- Ensure tests pass before merging changes
- Run tests locally when you conclude work on a feature or bugfix

## Fixing bugs

Every time a bug was identifed

- Add a failing test that demonstrates the bug (write test that will fail befor
  fix is applied (as expected) and will pass after the fix is applied)
- Fix the bug
- Ensure the test passes
